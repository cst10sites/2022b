What is CSS?

HTML controls the CONTENT (what is on the page) while CSS controls the DESIGN/LAYOUT (how it looks on the page). There are 3 types of CSS: inline CSS (from img style="width:25%;" in Lesson 1), internal CSS (in Lesson 2), external CSS (in Lesson 4).

Inline CSS

Inline CSS controls the design on PART of a tag. We used the span style="..." to control this behavior.

Internal CSS

Internal CSS controls the design on the ENTIRE page and ALL specific tags. We used the syle tag in the head section to control this behavior.
